🎮 Dark Escape 3D
Dark Escape 3D is a third-person adventure and survival game developed in C++ using the OpenGL graphics API. The game challenges players to navigate a dangerous floating path filled with dynamic obstacles and traps to reach the final goal.

📝 Overview
In this game, you control a 3D avatar navigating through a high-stakes environment. The gameplay is built on three core pillars:

1- Coin Collection: You must collect all 5 gold coins scattered across the map to unlock the "Goal Gate."

2- Hazard Avoidance: The path is rigged with moving spikes and oscillating laser beams that will reset your progress upon contact.

3- Precision Movement: Staying within the track boundaries is vital. Falling off the edges results in an immediate loss.

Upon losing, the game features a full state-reset: the player returns to the start, the coin counter is zeroed, and all collected coins reappear.

🕹️ Controls
Key              Action
W               Move Forward
S               Move Backward
A               Move Left
D               Move Right
Space           Jump
Left Shift      Sprint (Double Speed)
ESC             Quit Game
